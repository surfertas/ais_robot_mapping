ais_robot_mapping
=================



Usage
-----

From command line:

.. code-block::

    conda create -f environment.yml
    source activate robo_map
    cd robot_mapping
    python main.py


To test:

.. code-block::

    cd robot_mapping
    python -m pytest ../tests/
