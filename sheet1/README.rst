ais_robot_mapping
=================



Usage
-----

From command line:

.. code-block::

    conda env create -f environment.yml
    conda activate robo_map
    cd robot_mapping
    python main.py


To test:

.. code-block::

    cd robot_mapping
    python -m pytest ../tests/
